# MASK
Mints Automobile Sensing Kit 

# Devices Attached
- Rasberry Pi 2WH
- Pi Sugar 3 Power Manage
- Toggle Switch
- Sensors
  - IPS7100  : I2c or Uart
  - BME280   : I2c (0x77) 
  - COZIR AH : Uart Only 
  - BNO085   : I2c and Uart 
  - PA1010D  : I2c and Uart 
  - BME280   : I2c  
  - TMP117   : I2c
